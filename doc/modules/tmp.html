primers.santa_lucia (sequence, primer_concentration, salt_concentration, magnesium_concentration)
Returns the melting temperature of a primer given primer concentration, salt concentration, and magnesium concentration

Parameters:

  • sequence sequence of the primer
  • primer_concentration concentration of the primer

    melting temperature

Usage:

codon.ncbi_standard_to_codon_table (amino_acids, starts)
Generates a codon table from a list of amino acids and a list of start codons, given in the standard NCBI codon table format.

Parameters:

  • amino_acids the amino acid string
  • starts the start codon string

Returns:

    a codon table

Usage:

    local ct = codon.ncbi_standard_to_codon_table("FFLLSSSSYY**CC*WLLLLPPPPHHQQRRRRIIIMTTTTNNKKSSRRVVVVAAAADDEEGGGG", "---M------**--*----M------------MMMM---------------M------------")
